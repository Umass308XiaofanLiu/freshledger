# Specification deviations

Log deliberate departures from the canonical FreshLedger specifications here. The five publicly promised features and the food-safety conservatism rules are not eligible for deviation.

- 2026-07-17 — The implementation task runs in Codex Desktop (`gpt-5.6-sol`, agent core `0.145.0-alpha.18`) rather than the handoff's assumed standalone CLI 0.114.0; the standalone CLI installed on the machine is 0.116.0. Session evidence records both accurately.
- 2026-07-17 — Interim D0-to-D1 grounding state: until the full shelf-life CSV resolver replaces it, scan responses clamp LLM storage durations to `[1, 730]` and the specification's category maxima, use a conservative 3-day ceiling for unknown/unmapped categories, and emit the legal `llm_clamped` source; `storage_options` remains pending the D1 resolver.
- 2026-07-18 — The interim clamp above is now superseded by the 100-row reference resolver; exact/alias hits emit `reference` with location options, long-tail model guidance remains conservatively `llm_clamped`, and storage-less perishables use `default`.
- 2026-07-18 — Added an explicit zero-token `POST /v1/demo/receipts/{sample_id}/scan` path and `scan_provenance` response field. Saved synthetic samples enter the same safety, SQLite, review, and pantry pipeline but never call OpenAI; the separate Live route remains GPT-5.6-only and never silently falls back. This user-authorized mode keeps the public demo free and is permanently disclosed in the UI.
- 2026-07-18 — At the user's explicit direction, D1 persistence and Demo work proceeded while the canonical D0 real-GPT phone gate remains blocked by the absent `OPENAI_API_KEY`. Demo results are logged separately and are not counted as a Live GPT pass.

- 2026-07-18 — Demo meal suggestions and purchase-advice prose are deterministic, SQLite-derived, and permanently labeled `Demo · no AI call` rather than using the D2 GPT text calls. The complete user-visible meal and advice loops remain present, while the separate GPT-5.6 runtime path is preserved for future funded Live use.
- 2026-07-18 — The specification's 300-call global breaker is reduced to a configurable default of 3 OpenAI calls per UTC day because the owner requires a near-zero-spend build. The reservation is atomic in SQLite and is independent of per-IP limits; public Demo deployments leave the API key empty.

Fuzzy hints remain review-only with no canonical key; unmatched corrections reset to a conservative category/unknown default and cannot teach alias memory. This deliberate tightening prevents prepared or compound foods from inheriting unsafe substring advice.
- 2026-07-18 — Real-image cache keys now include the recognition pipeline version in addition to provider, model, and image SHA. Only draft records are cacheable; confirmed images return an already-imported conflict. This prevents stale parser/matcher results and duplicate confirmation behavior.
