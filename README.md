# FreshLedger

FreshLedger turns one grocery-receipt photo into an itemized spending ledger and freshness-aware food inventory.

This repository is the OpenAI Build Week 2026 implementation workspace. The D0 walking skeleton is an Expo React Native client backed by a Python FastAPI receipt-scanning service.

## Repository layout

- `app/` — Expo React Native application using React Native Paper.
- `server/` — Python 3.12 FastAPI service and OpenAI receipt-vision integration.
- `fixtures/receipts/` — standing receipt test images and expected parses.
- `docs/` — Codex session evidence, test logs, and specification deviations.

Setup and demo instructions will be added after the walking skeleton is verified.
